Project 1: Alzheimer's Disease Prediction

This project will be exploring if we can predict a patient's diagnosis of Alzheimer's based on cognitive assessment scores using an Alzheimer dataset. We will also explore if we can predict a cognitive exam score based on brain volume measurements.

Dataset Description:

The Alzheimer's dataset contains information from a research study comparing Patients with Alzheimer's disease and Healthy control subjects. It also contains features based on brain scans and cognitive test results for each patient.

Preparing our data and train logistic regression model
